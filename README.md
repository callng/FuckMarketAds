## 说明 / Desc:

小米应用商店 广告移除

- 一切开发旨在学习，请勿用于非法用途

---

## 功能 / Func

* 移除开屏广告
* 隐藏广告低栏
* 隐藏应用安全检测
* 禁止切换前台展示广告
* 将非正版APP重新显示到更新列表
* 显示被商店隐藏的更新
